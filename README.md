# 🕌 MasjidTV InfoScreen

**Islamic InfoTV CMS for Masjid, Surau & Home**  
Built with ❤️ by [Khalifah Territory](https://github.com/ixezranger)

## Tech Stack
React 18 · Vite · Tailwind CSS · Supabase · Cloudflare R2 · GSAP · Howler.js

## Features (in development)
- 🕌 Prayer times (Waktu Solat) from e-solat.gov.my
- 🖼️ GSAP LayerSlider for images, videos & YouTube
- 🎵 Audio library: Zikir, Quran recitation, Nasheed
- 📁 Google Drive import for media
- ⚙️ Full CMS admin panel
- 📱 TV / Tablet / Mobile preview switcher
- 🌙 Ottoman-Turkish Glassmorphism design

## Setup

```bash
git clone https://github.com/ixezranger/khalifah-territory-masjidtv-infoscreen.git
cd khalifah-territory-masjidtv-infoscreen
npm install
cp .env.example .env
npm run dev
```
